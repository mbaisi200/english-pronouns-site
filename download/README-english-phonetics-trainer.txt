# English Phonetics Trainer - Instruções de Instalação

## Como usar este app

Este é um web app feito com Next.js 15. Para rodá-lo, você precisa ter Node.js instalado.

### Passo 1: Extrair os arquivos
Extraia o arquivo ZIP em uma pasta no seu computador.

### Passo 2: Instalar dependências
Abra o terminal na pasta extraída e execute:

```bash
npm install
```

### Passo 3: Rodar o app
```bash
npm run dev
```

### Passo 4: Acessar no navegador
Abra http://localhost:3000 no seu navegador.

---

## Funcionalidades

- **7 vozes em inglês**: Emma, Sophie, Olivia, Charlotte (femininas) e James, Michael, David (masculinas)
- **Controle de velocidade**: De 0.5x a 2.0x
- **Responsivo**: Funciona em PCs, tablets e celulares
- **Compatível com iPhone**: Testado para funcionar em iOS

---

## Hospedagem Online (para compartilhar com alunos)

Para que seus alunos possam acessar, você precisa hospedar o app online:

### Opção 1: Vercel (gratuito e recomendado)
1. Crie uma conta em vercel.com
2. Instale: `npm i -g vercel`
3. Execute: `vercel` na pasta do projeto
4. Você receberá uma URL para compartilhar!

### Opção 2: Netlify
1. Crie uma conta em netlify.com
2. Arraste a pasta do projeto para o dashboard

## Suporte

Se tiver problemas, verifique se:
1. Node.js está instalado corretamente
2. Todas as dependências foram instaladas
3. A porta 3000 não está sendo usada por outro programa
